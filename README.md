# Advanced Customer Segmentation for E-commerce using Machine Learning

## 📌 Project Overview

This project focuses on intelligent customer segmentation for e-commerce businesses using machine learning and behavioral analytics.

Traditional RFM (Recency, Frequency, Monetary) analysis is enhanced with behavioral features such as:

- Session duration
- Purchase intervals
- Browsing patterns
- Engagement metrics
- Demographic enrichment

The system applies clustering algorithms like:

- K-Means
- DBSCAN

to identify hidden customer groups and generate business insights.
